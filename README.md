# SIM-Service-Intelligence-Matrix-
The Service Intelligence Matrix (SIM) is a decision support system designed to quantify the alignment between service advisor performance and customer expectations.
It transforms high-dimensional customer survey data into actionable insights through:

- Data reshaping (wide → long transformation)
- Correlation analysis (Spearman)
- Feature engineering (custom parsing & validation)
- Matrix-based scoring system (SA Match Score)

## 📊 Key Achievement

- Processed 37K+ surveys into 940K+ observations
- Identified key drivers of customer dissatisfaction
- Built a scoring system for advisor-customer matching
- Enabled data-driven workload allocation strategy
